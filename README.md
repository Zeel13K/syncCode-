## Follow these Instructions to Start the project:
1) Download it locally: <br>
   ``` git clone <repo_url> ```
2) Change the directory: <br>
``` cd <path_to_repo> ```
3) Install the dependencies: <br>
``` npm install ```
4) Start the Project: <br>
``` npm start ```
5) Usually the project will be on 'Localhost:5000'
